# 生活清单

生活清单与小红书模板、抖音模板、菜单同级，包括：

- 影视：电影、电视剧、动漫、动画、广播剧、小说；状态为想看、正在看、已阅；平台支持多选；支持分类内自定义排序。
- 分季内容：电视剧、动漫、动画、广播剧支持“作品 → 季/篇章 → 单集”层级。作品可标记值得重温/重听；季/篇章可保存独立封面；单集可填写标题和剧情、标记喜欢，并按作品分类搜索喜欢的单集。
- 活动：室内、户外、居家，仅记录文字项目。
- 行李：左侧为自定义场景，每个场景自动创建不可删除的“必备物品”层级，并可继续新增其他携带层级和物品。
- 吃什么：记录店铺、可外卖/可堂食属性以及可选菜品；用餐方式可以多选，不设置二级分类。

数据为全局共享。所有登录账号可读，只有 `WECHAT_ALLOWED_OPENIDS` 白名单账号可以新增、编辑和删除。

上线前在 Supabase SQL Editor 执行：

`supabase/migrations/202607110003_life_lists.sql`

分季与单集功能还需要继续执行：

`supabase/migrations/202607120005_media_seasons_and_episodes.sql`

季封面功能还需要继续执行：

`supabase/migrations/202607120006_media_season_covers.sql`
